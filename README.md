# Create — Creativity & Flow Library

A vibrant digital atelier of the **top 100 books** on creativity, flow state, craft, and making.

## Features

- **10 studios** (shelves) of curated literature
- Hover previews, book detail modals with essence + top 10 insights
- Real-world applications for creative life
- **Natural TTS** — listen to essence, insights, and applications
- Colorful, modern design for a space where creativity is born

## Local

```bash
cd ~/Desktop/Create
npx serve
```

## Stack

Static HTML, CSS, vanilla JS. Deploy on Vercel with no build step.
